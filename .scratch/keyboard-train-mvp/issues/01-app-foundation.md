# 01 — App foundation (scaffold + shell UI)

**Status:** resolved  
**Type:** task  
**Parent:** `.scratch/keyboard-train-mvp/spec.md` (`docs/PRD-mvp.md`)

## What to build

Scaffold aplikasi web **Keyboard Train** (Vite + React + TypeScript + Tailwind) yang bisa dijalankan lokal, dengan shell UI Bahasa Indonesia: judul app, layout bersih (clean modern edukasi), routing dasar, dan peringatan desktop/laptop + keyboard fisik pada viewport sempit.

Setelah issue ini selesai, developer/agent bisa `npm install && npm run dev`, melihat halaman awal berbahasa Indonesia, dan `npm run build` sukses — meski fitur latihan belum ada.

## Acceptance criteria

- [ ] Proyek Vite + React + TypeScript + Tailwind terinisialisasi di root repo
- [ ] `npm run dev` dan `npm run build` berhasil
- [ ] Shell UI berbahasa Indonesia (minimal: nama app, teks sambutan singkat)
- [ ] Client routing siap untuk halaman berikutnya (minimal home/welcome)
- [ ] Notice tampil pada viewport sempit: gunakan desktop/laptop dengan keyboard fisik
- [ ] README singkat: cara install & menjalankan dev server
